「近 2 年」AI 健康产品落地：GreenPlatter 与 Jovida

先在 GreenPlatter 做代谢与食谱测评方向的模型与内容运营，后在 Fluxvita 任北美 AI 减肥教练 Jovida 的产品侧；两段经历均为英文产品、合规语境下的 0–1 交付。

GreenPlatter

GreenPlatter Inc.｜美国 密歇根｜2024年10月 – 2025年7月

GreenPlatter · 应用图标

  • 基于循证医学，参与代谢综合征与糖尿病人群 AI 营养分析模型 0–1，控制糖尿病饮食下的营养素约束与配餐推送。
  • 参与构建 10000+ 食谱及测评集，模型推荐准确度与饮食多样性提高约 50%。
  • 独立开发健康科普图文批量生产与批量测试插件，测试与开发效率提升约 2–3 倍。
  • 参与 30+ 位糖尿病患者深访，持续优化推送体系与体验。

Jovida

Fluxvita｜北京 海淀｜2025年5月 – 至今

Jovida · App Store 图标

团队合影

  • 面向北美市场的 AI 减肥教练 Jovida：0–1 业务架构，参与构建「Agent-in-Human-Loop」交互；将临床诊断思维转化为 Agent 决策树，保障医疗级输出准确性。
  • 主导能量燃烧、食物评分体系、外食与菜谱推荐、Onboarding 等模块产品侧全流程 0–1：调研、agent 设计、工作流 demo、skill 与 prompt 落地、验收与埋点。次日留存超过 25%，核心功能使用率超过 70%，核心步骤转化率超过 90%。
  • Vibe Coding 全栈高保真 demo 与 API/MCP demo，提升与研发的交接效率。
  • 构建专属大模型评测集，带领团队 prompt 调试，营养素识别与食物评分精准度提高约 30%。
  • 独立运营 Instagram 与 TikTok，实现从 0 到 100 的涨粉突破。
